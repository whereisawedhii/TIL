## 할당문

1. 할당 연산자 = 오른쪽에 있는 표현식을 평가해서 값 (메모리 주소) 생성 
2. 값의 메모리 주소를 = 왼쪽에 있는 변수에 저장 
- 존재하지 않는 변수라면 
  - 새 변수를 생성 
- 기존에 존재했던 변수라면
  - 기존 변수를 재사용해서 변수에 들어있는 메모리 주소를 변경

```
number = 10
double = 2 * number
print(double)

number = 5
print(double)
```
number 가 바라보는 값만 바뀜 
double 이 바라보는 값은 바뀌지 않음 (재할당되지 않았음)

## Sequence type 
여러 개의 값들을 순서대로 나열하여 저장하는 자료형 

데이터들이 본인의 위치에 대한 순서가 존재하고, 순서대로 나열되어 있다 

*str, list, tuple, range*

1. 순서 sequence : 값들이 순서대로 저장 (정렬 X)
2. 인덱싱 indexing : 각 값에 고유한 인덱스 (번호)를 가지고 있으며, 인덱스를 사용하여 특정 위치의 값을 선택하거나 수정할 수 있음 
3. 슬라이싱 slicing : 인덱스 범위를 조절해 부분적인 값을 추출할 수 있음 
4. 길이 length : `len()` 함수를 사용하여 저장된 값의 개수(길이)를 구할 수 있음
5. 반복 iteration : 반복문을 사용하여 저장된 값들을 반복적으로 처리할 수 있음

## 슬라이싱 slicing
시퀀스의 일부분을 선택하여 추출하는 작업

시작 인덱스와 끝 인덱스를 지정하여 해당 범위의 값을 포함하는 새로운 시퀀스를 생성 

문자와 문자 사이 중간 빈 공간을 자른다고 생각해봐 

```
my_str = 'hello'
# 슬라이싱 
print(my_str[2:4])
# 시작 생략, 끝 지정 : 처음부터 시작 
print(my_str[:3])
# 시작 지정, 끝 생략 : 끝까지 
print(my_str[3:])
# step을 지정하여 추출 
print(my_str[0:5:2])
# 문자열 뒤집기 
print(my_str[::-1])

```
궁금한게 있으면 먼저 손으로 써보고 결과를 보고 혼자 생각해봐
